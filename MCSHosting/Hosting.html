1단계: 자바(Java) 설치 확인
마인크래프트 서버는 자바 기반으로 돌아갑니다. 현재 플레이하려는 버전과 맞는 자바가 설치되어 있어야 합니다.

최신 버전(1.20.x 이상): Java 17 또는 Java 21이 필요합니다.

확인 방법: 명령 프롬프트(CMD)를 열고 java -version을 입력했을 때 버전 정보가 나오면 통과입니다.

2단계: 서버 파일 다운로드 및 실행
공식 사이트 접속: 마인크래프트 공식 서버 다운로드 페이지에서 minecraft_server.1.x.x.jar 파일을 받습니다.

폴더 생성: 바탕화면 등에 새 폴더(예: MC_Server)를 만들고 다운로드한 파일을 넣습니다.

첫 실행: 파일을 더블 클릭하거나, CMD에서 해당 폴더로 이동해 아래 명령어를 입력합니다.

java -Xmx2G -Xms2G -jar server.jar nogui
(2G는 서버에 할당할 램 용량입니다. 여유가 있다면 4G로 늘려보세요.)

3단계: EULA 동의 (필수)
처음 실행하면 폴더에 몇 가지 파일이 생기면서 서버가 곧바로 꺼질 겁니다.

생성된 eula.txt 파일을 메모장으로 엽니다.

eula=false 부분을 eula=true로 수정하고 저장합니다.

다시 서버 파일을 실행하면 이제 서버 데이터(맵)가 생성되며 정상 구동됩니다.

4-1단계: 내 컴퓨터의 '내부 IP' 확인하기
공유기에게 "누구 컴퓨터로 연결해줄까?"라고 물었을 때 대답할 주소가 필요합니다.

윈도우 키 + R을 누르고 cmd를 입력해 실행합니다.

검은 창에 ipconfig를 입력하고 엔터를 누릅니다.

IPv4 주소 항목의 숫자(예: 192.168.0.7)를 따로 적어두세요. 이것이 내 컴퓨터의 내부 주소입니다.

기본 게이트웨이 주(예: 192.168.0.1)도 확인하세요. 이것이 공유기 설정 페이지 주소입니다.

4-2단계: 공유기 관리 페이지 접속
인터넷 브라우저(크롬, 엣지 등)를 엽니다.

주소창에 위에서 확인한 기본 게이트웨이 주소를 입력하고 접속합니다.

로그인 화면이 나옵니다. (초기 아이디/비번은 보통 admin/admin이나 공유기 스티커에 적혀 있습니다.)

4-3단계: 포트포워딩 설정 (핵심)
공유기 제조사마다 메뉴 이름이 조금씩 다르지만, 보통 [고급 설정] -> [NAT/라우터 관리] -> [포트포워딩 설정]에 있습니다.

아래와 같이 새 규칙을 추가하세요:

규칙 이름: Minecraft (자유롭게 입력)

내부 IP 주소: 아까 적어둔 본인의 IPv4 주소

프로토콜: TCP (또는 TCP/UDP 둘 다)

외부 포트: 25565 ~ 25565

내부 포트: 25565 ~ 25565

설정 후 [추가] 또는 [적용]을 누릅니다.

4-4단계: 친구에게 알려줄 '공인 IP' 확인
이제 대문은 열렸습니다. 친구들에게 우리 집 주소를 알려줘야 합니다.

네이버나 구글에 '내 IP 확인'이라고 검색합니다.

나오는 숫자 주소(예: 211.xxx.xxx.xxx)가 바로 친구들이 접속할 주소입니다.

친구들에게 이 주소를 알려주고, 마인크래프트 [직접 연결]창에 입력하라고 하세요.

4-5단계: 안 될 때 체크리스트
포트포워딩을 다 했는데도 친구가 못 들어온다면, 윈도우 자체 방화벽이 막고 있을 확률이 높습니다.

제어판 > 시스템 및 보안 > Windows Defender 방화벽 > 허용되는 앱 설정으로 이동합니다.

Java(TM) Platform SE binary 항목을 찾아 개인/공용 모두 체크되어 있는지 확인하세요.

없다면 다른 앱 허용을 눌러 서버 구동기(server.jar)를 실행하는 Java 경로를 추가해야 합니다.
